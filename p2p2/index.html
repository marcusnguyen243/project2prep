Libraries In Use

For project two, I have decided to use p5.js and p5.play as the two main libraries. I've settled on these two libraries because the goal of my project is to create a game-like emulative experience of a zoo. p5.js will give me the overall game engine for my project and p5.play will give me the sprites and interaction that I need for this project. I will also be using p5.SceneManager to manage different scenes of the zoo.

Features

The three main features that I will be using for this project are p5.play's createSprite(), animation(), and changeAnimation(). These will be used to create the sprites of animals in my zoo.
